# Change Log

All notable changes to the "Open Recent Projects / Folders (Quick access)" extension will be documented in this file.

## [0.1.4] - 2026-02-12

### Added

- Initial release: Open Recent Projects / Folders (Quick access).
- Command "Open Recent Projects / Folders (Quick access)..." with keyboard shortcut `Alt+R` (Windows/Linux) / `Option+R` (macOS).
- Reads recent paths from Cursor/VS Code SQLite state DB.
- Fallback to sql.js when system sqlite3 is not available (e.g. on Windows).
